# Finance Dashboard

A sleek financial dashboard to track income, expenses, and insights. Built with React and Tailwind CSS.

## Features

- **Dashboard Overview** – Balance cards, 7-day trend chart, spending breakdown by category
- **Transactions** – Searchable list with add/edit (admin only)
- **Role-Based Access** – Switch between Viewer (read-only) and Admin roles
- **Insights** – Highest spending category, month-over-month comparison, financial health tips
- **Responsive Design** – Works on mobile, tablet, and desktop
- **Clean UI** – Professional dark theme with intuitive layout

## Tech Stack

- **Frontend Framework**: React 18
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **State Management**: React Context API
- **Language**: JavaScript (ES6+)

## Setup

### Requirements
- Node.js v16+
- npm v8+

### Installation
```bash
npm install
npm run dev
```
Then open `http://localhost:5173/`

## Usage

**Viewer** – See all data, search transactions  
**Admin** – Add/edit transactions, plus all viewer features

Switch roles using the dropdown at the top of the dashboard.
